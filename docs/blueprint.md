# **App Name**: Splash

## Core Features:

- VLF-Tec Splash Sequence: An immersive 2-second animated introduction featuring the VLF-Tec branding upon application launch.
- Firebase Auth Integration: Secure login system utilizing Firebase Realtime Database to manage user credentials and persistent sessions.
- Restricted Account Registration: A sign-up process that captures username, email, and a unique Chat Name while enforcing character rules like the exclusion of the '@' symbol.
- Account Creation Throttling: A security feature that limits users to creating one account every 2 hours to prevent system abuse.
- Intelligent Form Persistence: Automatically remembers and pre-fills username and email fields for returning users to streamline the login experience.
- Sophisticated App Dashboard: A stylish main menu displaying the 'Splash' brand identity with advanced typography and layout components.

## Style Guidelines:

- A sleek dark-mode palette using Charcoal Black (#121212) for the background, Deep Indigo (#1A237E) for primary elements, and Neon Cyan (#00E5FF) for interactive accents.
- Uses 'Montserrat' for bold, impactful headers and 'Inter' for clean, highly legible body text and form fields.
- Minimalist, thin-stroke icons with a faint cyan glow to indicate interactivity and status.
- A balanced, centered approach for authentication screens and a modern grid-based main menu emphasizing white space and visual clarity.
- Smooth, high-fidelity transitions between screens and a dedicated 2-second splash animation using easing curves for a premium tech aesthetic.
